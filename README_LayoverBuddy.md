# LayoverBuddy Landing Page

A beautiful, modern landing page for LayoverBuddy - an app that helps travelers make the most of their layovers.

## Features

✈️ **Modern Design**: Clean, responsive design with smooth animations
🎨 **Beautiful UI**: Gradient backgrounds, hover effects, and smooth transitions
📱 **Mobile Responsive**: Fully responsive design that works on all devices
🚀 **Interactive Elements**: Scroll-triggered animations and smooth navigation
⚡ **Fast Loading**: Optimized performance with minimal external dependencies

## What LayoverBuddy Offers

The landing page showcases an app concept that provides:

- **Nearby Attractions**: Discover museums, parks, and tourist spots within layover time
- **Local Cuisine Finder**: Find restaurants and local food experiences near airports
- **Lounge Access**: Book airport lounges and find comfortable rest areas
- **Smart Time Management**: AI-powered itinerary planning that ensures you don't miss flights
- **Transportation Guide**: Real-time info on shuttles, taxis, and public transport
- **Safety & Security**: Vetted recommendations with emergency contacts

## How to View

1. **Open directly**: Simply open `index.html` in any modern web browser
2. **Local server**: Run `python3 -m http.server 8000` and visit `http://localhost:8000`

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with gradients, animations, and responsive design
- **JavaScript**: Smooth scrolling, intersection observer for animations, and interactive elements
- **Font Awesome**: Icons for visual enhancement
- **CSS Grid & Flexbox**: Modern layout techniques
